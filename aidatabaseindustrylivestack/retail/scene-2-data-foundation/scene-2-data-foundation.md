# Scene 2 Data Foundation

## Introduction

This scene prepares the Seer Sporting Goods dataset that powers the rest of the LiveStack Demo. The page loads or restores the governed Oracle AI Database 26ai data foundation, then shows the main data domains and data types used across the retail use cases.

Estimated Time: 7 minutes

![Data Foundation page with the load action, data carousel, and Oracle Internals sidebar highlighted](images/data-foundation.png)

### Objectives

In this scene, you will:
- Load the Seer Sporting Goods demo dataset, or restore it if data is already loaded.
- Understand what data is prepared for the LiveStack Demo.
- Review the **What Gets Loaded** carousel to see the data domains and data types used across the demo.
- Use the expandable **Oracle Internals** sidebar to connect the page to the Oracle AI Database features behind it.
- Continue to the next scene after the dataset is ready.

## Task 1: Prepare the dataset

1. Click **Data Foundation** in the sidebar.
2. In **Prepare the Dataset**, click **Load Demo Data**.
3. If the dataset is already loaded, click **Restore Demo Data** to reset the demo data to a clean state.
4. Wait for the progress indicator to finish and review the refreshed record counts.

The load prepares products, customers, orders, returns, customer signals, fulfillment geography, vector embeddings, machine learning outputs, and agent audit history. These records become the shared foundation for every later scene.

## Task 2: Review what gets loaded

1. Scroll to **What Gets Loaded**.
2. Read the three visible carousel tiles.
3. Use the right carousel arrow to review the remaining tiles.
4. Look for the major data types used in the demo: relational tables, JSON duality views, graph relationships, spatial geometry, vector embeddings, machine learning results, and agent audit records.
5. Expand or review the **Oracle Internals** sidebar on the right. It summarizes why the page matters, which Oracle AI Database capabilities are being used, and which tables or data structures support the scene.

You can move to the next scene.
