# Retail Returns Intelligence LiveStack Guide

## Introduction

This workshop is the runbook for the Retail Returns Intelligence LiveStack. It walks a presenter or customer through a retail operating model where return risk, social demand signals, creator influence, fulfillment routing, order documents, in-database machine learning, natural-language SQL, and AI agent workflows all run from one governed Oracle AI Database 26ai foundation.

Estimated Demo Time: 90 minutes

![Retail LiveStack welcome page](images/welcome-and-demo-orientation.png)

### Objectives

In this workshop, you will:
- Orient the audience to the retail business story and the seeded PeakGear demo data.
- Navigate each visible LiveStack scene using the application sidebar.
- Show how Oracle AI Database 26ai supports relational, JSON, graph, vector, spatial, ML, and AI agent patterns in one application.
- Use screenshots and expected outcomes to keep the walkthrough tied to what users see on screen.
- Run or replay the portable LiveStack package with Podman Compose.

### Prerequisites

This workshop assumes you have:
- Access to the running Retail Returns Intelligence LiveStack.
- A modern browser open to the application URL.
- The portable package `livestack-retail.zip` if you want to run the stack locally.
- Podman and Podman Compose for the download lab.
- Network access to the configured app port, which defaults to `http://localhost:8505`.

## Workshop Flow

- Download and run the LiveStack package.
