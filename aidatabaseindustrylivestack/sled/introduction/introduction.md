# State and Local Government Service Operations LiveStack

## Introduction

This workshop is the demo runbook for the State and Local Government Service Operations LiveStack. The application shows how Oracle AI Database 26ai can support resident service coordination from one governed data foundation: relational service requests, JSON duality views, resident demand signals, vector search, spatial coverage, graph relationships, Oracle Machine Learning, natural-language SQL, and agent-assisted operations.

Estimated Demo Time: 1 hour 45 minutes

![SLED Operations Brief home screen](images/sled-operations-brief.png)

### Objectives

In this workshop, you will:
- Navigate the LiveStack scene by scene using the application sidebar and top-bar actions.
- Connect visible application behavior to Oracle-backed data services, including JSON, graph, vector, spatial, OML, ORDS, and agent audit trails.
- Run the main operator interaction in each scene and explain what changes on screen.
- Use the download lab to run the portable LiveStack bundle with Podman Compose.
- Capture the business story for a state or local government team that needs better service access, demand sensing, and governed AI operations.

### Prerequisites

This workshop assumes you have:
- Access to a running SLED LiveStack application.
- A modern browser open to the application.
- Basic familiarity with public service operations, resident demand signals, service capacity, and cross-agency coordination.
- For the download lab only, Podman and Podman Compose available on the target machine.

## Workshop Flow

- Download and run the LiveStack locally, or connect to the hosted SLED instance.
- Scene 1: SLED Operations Brief.
- Scene 2: Seer 26ai Data Foundation.
- Scene 3: Public Service Command Center.
- Scene 4: Resident Demand Signals.
- Scene 5: Community Partner Network.
- Scene 6: Service Access and Coverage Map.
- Scene 7: Service Request Workbench.
- Scene 8: Demand and Capacity Analytics.
- Scene 9: Ask Seer Operations Data.
- Scene 10: Public Service AI Agent Console.
- Scene 11: Use Your Own Public Service Data.
- Conclusion and business outcomes.
